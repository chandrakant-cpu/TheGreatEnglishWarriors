# TheGreatEnglishWarriors

A comprehensive English learning platform designed like a military academy, where warriors train to master English from beginner to advanced levels.

## 🚀 Features

- **Three-Level Curriculum**: Beginner (Foundation Patrol), Intermediate (Communication Command), Advanced (Leadership Language)
- **Military-Themed Design**: Inspired by joining the Indian Navy, with disciplined training approach
- **Interactive Learning**: Live sessions, progress tracking, and achievement badges
- **Detailed Course Pages**: Comprehensive curriculum for each level
- **Progress Dashboard**: Track learning journey and achievements
- **Responsive Design**: Works on all devices

## 📚 Course Levels

### 🏕️ Foundation Patrol (Beginner - 8 Weeks)
- Basic grammar and vocabulary
- Everyday communication skills
- Pronunciation training
- Simple conversation practice

### 🎯 Communication Command (Intermediate - 12 Weeks)
- Advanced grammar structures
- Professional communication
- Public speaking and presentations
- Business English skills

### 👑 Leadership Language (Advanced - 16 Weeks)
- Executive-level communication
- Academic and professional writing
- Global business English
- Leadership and negotiation skills

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **Vanilla JavaScript**: Interactive features and animations
- **Google Fonts**: Inter font family for clean typography

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Local web server (optional, for better experience)

### Running Locally

1. **Clone or download** the project files
2. **Navigate** to the project directory
3. **Start a local server**:
   ```bash
   # Using Python (recommended)
   python -m http.server 8000

   # Or using Node.js
   npx serve .

   # Or using PHP
   php -S localhost:8000
   ```
4. **Open your browser** and go to `http://localhost:8000`
